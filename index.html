Your personal live Internet Dashboard

Focus Board is a simple tool that allows you to visually scrape interesting parts of different websites to create you personal internet dashboard.

Features:

  • Capture a visual area from any website
  • Display captured areas on an interactive dashboard
  • Use high-speed VPS for refreshing captured areas concurrently
  • Dispatch custom clicks to interact with websites JavaScript
  • Hide ads/popups using jQuery selectors
